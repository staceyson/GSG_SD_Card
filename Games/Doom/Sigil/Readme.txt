Wad files need a corresponding (empty) zip file so the SD card browser can "find" the games

The ATARI.WAD requires DOOM2.WAD, DOOM1.WAD is the shareware version of Doom.
Sigil 1 and 2 require a copy of DOOM.WAD in the Sigil folder with them in order to run

Buttons:

Dpad - movement
X - use
A - shoot
Y - run
B - strafe
L1 + R1 - strafe left/right
L2 + R2 - cycle weapons
Credit - map
Start - menu
Select - 180 degree turn

Note: Selecting Quit in-game will take you to a broken settings menu, just press the home button to exit after saving
