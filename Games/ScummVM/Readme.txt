ScummVM

To launch scummvm, select the scummvm.zip in the SD card browser menu

Place game folders in here, Add them like you normally would within ScummVM. If you select Quit from the ScummVM menu, you will end up at a broken GSG settings menu, just press the Home button to return to the SD card browser menu

Dpad and Trackball control the pointer
A & B are left and right mouse button
X button is F5
Y is Esc(ape)
L1 is the Space button, R1 is Enter
R2 hold to slow down the mouse cursor
Start brings up the ScummVM menu
